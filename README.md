
# DI Desktop Player

## Supports

- DI.FM
- RadioTunes
- Zen Radio
- Rock Radio
- Classical Radio
- Jazz Radio

## Linux

### Install dependencies

```
sudo dnf install gtk3-devel webkit2gtk4.1-devel
```

### Notes

Currently requires `WEBKIT_DISABLE_DMABUF_RENDERER=1` under Wayland
